# Heroku .Net Core Buildpack Sample Code
## by Softtrends LLC
### This Sample code uses .NET Core 2.1 SDK and Libraries

This Sample code uses the .Net Core 2.1 Buildpack provided by Softtends and adds full support for Heroku including the Postgres library referenced in the project.json file. For other .Net core see the respective sample code

*   Sample pages using ASP.NET Core 2.1 MVC
